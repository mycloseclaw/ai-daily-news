# 🤖 AI Daily News

每日 AI 资讯聚合，自动更新。

## 简介

每天早上自动抓取 AI 领域的最新资讯，包括：
- OpenAI、Anthropic、Google DeepMind 等大厂动态
- AI 研究论文和突破
- 行业趋势和应用

## 技术栈

- **前端**: 纯 HTML + CSS + JavaScript
- **数据源**: RSS 订阅 + 公开 API
- **自动化**: GitHub Actions 每日定时抓取
- **部署**: GitHub Pages

## 自动化工作流

### GitHub Actions

项目已配置 GitHub Actions 自动化：

- **每日定时更新**: 每天凌晨 2:00 自动抓取最新新闻
- **手动触发**: 支持手动运行自动化工作流
- **分支保护**: 确保主分支代码质量

### 工作流配置

工作流文件位于 `.github/workflows/daily-update.yml`，包含：

**执行步骤：**
1. 检出代码
2. 安装 Python 依赖
3. 运行 `fetch_news.py` 抓取新闻
4. 提交更新到 `news.json`
5. 自动推送到远程仓库

### 2. 核心功能模块

#### 2.1 新闻展示模块 (news.js)

**功能：**
- 加载并解析 `news.json` 数据
- 支持按标题关键词搜索
- 支持按日期筛选
- 分页加载（可选）
- 新闻卡片展示

**数据结构 (news.json):**
```json
{
  "lastUpdated": "2026-03-15T10:00:00Z",
  "news": [
    {
      "id": "unique-id",
      "title": "新闻标题",
      "source": "来源",
      "url": "原文链接",
      "date": "2026-03-15",
      "summary": "摘要内容",
      "tags": ["AI", "Microsoft"]
    }
  ]
}
```

#### 2.2 用户认证模块 (auth.js)

**功能：**
- 用户注册（用户名 + 密码）
- 用户登录验证
- Session 管理（localStorage）
- 登出功能

**用户数据存储 (users.json):**
```json
{
  "users": [
    {
      "username": "user1",
      "password": "hashed_password",
      "createdAt": "2026-03-15T10:00:00Z"
    }
  ]
}
```

**安全说明：**
- 密码使用 SHA-256 哈希存储
- 前端简单验证，生产环境建议使用后端服务

#### 2.3 留言板模块 (board.js)

**功能：**
- 展示留言列表
- 发表评论
- 显示评论时间
- 需要登录后才能发言

**留言数据结构：**
```json
{
  "messages": [
    {
      "id": "msg-id",
      "username": "user1",
      "content": "评论内容",
      "timestamp": "2026-03-15T10:30:00Z"
    }
  ]
}
```

#### 2.4 搜索与筛选

**搜索功能：**
- 实时搜索（输入时触发）
- 搜索标题和摘要
- 大小写不敏感

**日期筛选：**
- 日期选择器
- 显示选定日期的新闻
- 支持清除筛选
